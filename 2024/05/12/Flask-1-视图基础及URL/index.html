查询参数的使用

网址中在 ? 后面的格式为 key=value 的以 & 作为分割的字符串称为查询参数。

为何叫查询参数？因为无论怎么加查询参数（有用的或者没用的）都不会改变所调用的 API 接口，而这些查询参数会传递到 API 接口内部，就像给函数传递参数一样。

此时在浏览器中输入 http://127.0.0.1:5000/?uname=zhuning&pwd=1234后台就可以获取到参数 uname 和 pwd。

查询参数是直接暴露在 url 地址中的，用查询参数发起的请求称为 get 请求。

请求体参数

当请求的数据需要隐藏或者数据量较大时，就需要使用请求体参数。

请求体参数是隐藏在数据包中的，使用请求体参数发起的请求称为 post 请求。

注意在浏览器中输入地址无法发送 POST 请求，此时需要使用 PostMan 模拟 POST 请求。
